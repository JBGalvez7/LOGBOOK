Export Month to Spreadsheet

If the Google Spreadsheet URL is saved in Settings, this downloads the connected Google Sheet directly — which has entries from all connected devices. If offline or no Spreadsheet URL is set, it generates a local Excel file from this device only.

To open in Google Sheets: go to Google Drive → New → File upload, select the downloaded file. Right-click → Open with → Google Sheets.

Employee Registry

Employees added here can just type their ID on the New Entry form and their details will fill in on their own. Their signature gets saved the first time they submit an entry.

Google Sheets Integration

The system syncs every entry to Google Sheets automatically. Both fields below are needed — the Script URL for sending entries, and the Spreadsheet URL for downloading combined exports.

Data Backup & Restore

Download a copy of all logbook entries as a backup file. It's a good idea to save this somewhere safe like a USB drive or Google Drive in case anything happens to the device.
